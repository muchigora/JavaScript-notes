Populating Arrays

Indexes

  • You can access an array by referring to an index position you want to add the elemnt to
  • if an element already exists at that index position, it will be overwritten
  • using the length property allows us to add item to the end of the array

Adding Batman movies to index position 1

let movies = ['Caddyshack', 'Interstellar', 'Scarface'];
