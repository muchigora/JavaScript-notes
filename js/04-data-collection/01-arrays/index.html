Arrays Intro

Syntax:

let array = [item-0, item-1, item-2, item-3]
let cars = ["Saab", "Volvo", "Audi", "BMW", "Mercedes", "Chevrolet", "]

What are Arrays?

  • A special type of variable that can hold more than one data value
  • Technically, arrays in JS are objects , made up of a list with keys of "0", "1", etc called indexes, first index is [o]:
    • Arrays are a special type of objects. The typeof operator in JavaScript returns "object" for arrays.
  • Contain elements, which can be 0 or more
  • Each element in and array can hold any data types including objects, functions, even other arrays.
  • Arrays elements are ordered
  • Good practice is to name arrays variables in plural form, e.g., let colors = ['red', 'green', 'blue'];
