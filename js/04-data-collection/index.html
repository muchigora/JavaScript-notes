Data Collections

Data collections:

  • allow you to organize many different pieces of data into a single variable
  • A Variable is a / points to a memory location where a value for the variable is stored
  • in the code snippet below, the variable x is assigned the value of 10
    let x = 10;
    JavaScript assigns a memory location to store the value 10 and associates it with the variable x.
  • Data Collections come in two forms:
    1. indexed:
      • organize their contents in a specified order, with a sequential numerical index for each element in the collection.
      • example: array,
    2. keyed:
      • organize their contents with descriptive keys that are linked to values in key:value pairs.
      • example: objects
