What is an Iterator?

an iterator is an object that can be used to loop though iterable sata, like strings, arrays, etc

an iterator is an object that provides a way to access the elements of a container or collection sequentially, without exposing the underlying implementation details of the container

It allows you to iterate or loop through the elements of a collection, such as an array, list, or set, in a specific order.

The concept of iterators is closely associated with the iterator design pattern, which separates the algorithm for traversing a collection from the collection itself. By using iterators, you can iterate over a collection without worrying about its internal structure or representation.

let myArray = [1, 2, 3, 4, 5];
let iterator = myArray[Symbol.iterator](); // Create an iterator from the array
// Iterate over the elements using a loop
for (let element of iterator) {
console.log(element);
}
