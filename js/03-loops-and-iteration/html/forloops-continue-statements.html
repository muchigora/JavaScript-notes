for...in Loop

The for...in statement iterates through a set of enumerable properties of an object using a specified variable
For each distinct property, JavaScript executes the statements defined within the loop.

// for of loop looping though objects:
const person = {firstName: 'John',
lastName: 'Doe',
age: 30
};
let value = "";
for(let key in person){
value += person[key] + " ";
console.log(value)
}

result code

Example Notes

  • the for in loop iterates over a person object
  • each iteration returns a key
  • each key is used to access the value of the key
  • the value of the key is person[key]
