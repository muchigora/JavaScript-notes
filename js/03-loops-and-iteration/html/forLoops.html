for Loops

For loops are typically used to perform a task a set number of times.
A for loop will repeat until a specified condition evaluates to false

for (initialization; condition; afterthrough) {
statements

Steps:

for(let i=0; i<10; i++){
console.log(i);
}
  1. initialization expression or initialization: let i = 0;
    • initializes one or more loop counters
    • the syntax allows an expression of any degree of complexity
    • can also declare variables
    • initialization let i = 0; is executed before the loop starts
  2. the condition exppression is evaluted: i < 10;
    • defines the confition for running the loop
    • if condition is true, the loop statements executes
    • if false the loop terminates
  3. afterthought is executed, if present i++
    • executed after each time the loop has been executed
  4. control returns to step 2 i < 10

The for loop tests the condition given before the execution of each loop. As long as it evaluates to true, the loop will run again. When the condition evaluates to false, the loop exits and the program continues. At the end of each loop, the for loop executes the afterthought, which should push the loop further towards its conditional statement returning false.
